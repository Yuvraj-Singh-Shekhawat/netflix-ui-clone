# 🎬 Netflix Clone

A responsive Netflix-inspired frontend clone built using HTML, CSS, and JavaScript. This project recreates the modern Netflix user interface with a focus on clean design, responsiveness, and user experience.

## 🔗 Live Demo
https://frontui-clone.netlify.app/

## 📌 Features

* Netflix-inspired UI Design
* Responsive Layout for Mobile and Desktop
* Hero Banner Section
* Movie & TV Show Categories
* Hover Effects and Animations
* Smooth User Experience
* Clean and Modern Interface


## 🎯 Learning Outcomes

* Responsive Web Design
* CSS Flexbox and Grid
* JavaScript DOM Manipulation
* UI/UX Design Principles
* Frontend Project Deployment
